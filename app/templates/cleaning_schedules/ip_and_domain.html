Ip and Domain

Why using an ip address but not some normal words like goole.com?

Ultimately, we do need ip address, the series of numbers, to access a website.

We can access google.com with words instead of something like 142.250.179.174 because we have DNS, Domain Name System, a bridge between the name and numbers.

However, this is a local website, meaning you can only access this website if you are connecting to the house's wifi, so a public DNS server obviously has no access to it and thus cannot help you with it. And even if it can access the local server, someone need to pay for the service, who is definitely not me.

What alternatives do we have?

We do not have a dns server, but we do have a router that can serve similar functionality. Unfortunately, no one know the administratior password.

The most simple method is to let your browser remember the ip so it can auto-complete the address for you.

If you are using a laptop or a pc, you can also modify the "hosts" file to set a customed name for the ip address, so you only need to type in the customed name next time you need to access the website

Although it is probably possible for me to setup a local dns server, you would still need to manually config the dns on all your devices since, still, no one can access the router settings. So, i guess it is not worth the trouble.
